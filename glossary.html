Glossary

Elements
Defined by a start tag, some content, and an end tag.
Empty elements do not have end tags!
Attributes
Provide additional information about Elements
Come in name/value pairs such as name="value"
Favicon
The icon present in the browser tab next to your web page title
HTML Comments
HTML comments are used for documenting HTML, helps with readability or explains non-obvious features of HTML code
JavaScript
Programming language that allows for more dynamic and interactive web pages, utilized by the "script" tag
