As demonstrated above, the "div" tag is primarily for the purposes of applying
CSS styles to block elements, where as "span" is for inline elements kind of like this as a more subtle example.
This is done primarily through class tags so different div/span tags can be flexibly applied.

also be used to further customize CSS style options in div and span elements.
